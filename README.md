# 猿恋的个人图床
用于博客和其他地方。



### 命名规则

图片命名按照时间戳+图片名称拼音，方便后期查找。

例如：202205102256582-ceshi.jpg

拼音不太好：[在线汉字转换拼音工具](https://www.aies.cn/pinyin.htm)

### 使用
https://gcore.jsdelivr.net/gh/[Github名]/[仓库名]/[文件夹/文件名]
